# fakestore-react-shop

Application e-commerce développée avec **React JS** consommant l’API publique [FakeStoreAPI](https://fakestoreapi.com/).
Ce projet a pour objectif pédagogique l’apprentissage de React, du travail collaboratif avec Git et de l’architecture frontend moderne.


## Fonctionnalités

* Authentification utilisateur
* Consultation des produits
* Détails d’un produit
* Gestion du panier
* Suivi des commandes
* Profil utilisateur
* Historique des achats

## Objectifs pédagogiques

* Comprendre l’architecture d’une application React
* Consommer une API REST
* Gérer les états globaux
* Collaborer avec Git & GitHub
* Structurer un projet professionnel

---

## Stack technique

* React JS
* React Router
* Context API / Redux
* TailwindCSS
* FakeStoreAPI

## ⚙️ Installation

```bash
git clone https://github.com/laurence1Mas/fakestore-react-shop.git
cd shopverse-react
npm install
npm run dev
```
